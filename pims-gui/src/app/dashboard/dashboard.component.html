About The PIMS Project

The Population Information Management System (PIMS) Project is a CSC 312: Database Design project with the aim of simulating the storage and retrieval of information regarding a large population of people. Our database stores randomly generated population data for 1,000,000 'people' and we have used this project to practice writing queries to retrieve relevant information about that population.

Our API

Our project used a Java Spring API as our backend. The Java Spring framework is an open-source framework that streamlines the process of creating a RESTful API. Using this technology we were able to create an easily extendable API that allowed us to impliment custom queries to our database and serve that information to our Angular front-end application through a JSON payload retrieved via http requests.

Our Angular Front End

AngularJS is a google maintained open-source framework that facilitates single-page web applications. Using AngularJS we were able to create a web based front end for our database application where we could display tables and graphs that depict statistical information about our population that was retrieved from the database by querying our API through an http client.

Our Database

This is our database design, we have 1,000,000 population members and each of them have location, health, and occupation data. We also have 500 businesses that the members of out population work for, and these sets of data are connected via the occupation table.
